# 📌 Registro de Mascotas

Aplicación web desarrollada con HTML, CSS y JavaScript que permite registrar mascotas mediante su nombre y número de chip. La información introducida por el usuario se muestra dinámicamente en una lista, ofreciendo herramientas básicas para organizar y gestionar los registros almacenados.

## Descripción

Este proyecto fue desarrollado con el objetivo de practicar conceptos fundamentales del desarrollo web frontend, especialmente la manipulación del DOM, la gestión de eventos y el trabajo con estructuras de datos en JavaScript.

La aplicación permite añadir nuevas mascotas a una lista mediante un formulario sencillo. Una vez registrados, los datos pueden visualizarse de forma inmediata, ordenarse según los criterios implementados o eliminarse completamente cuando sea necesario.

## Funcionalidades

- Registro de mascotas mediante nombre y número de chip.
- Generación dinámica de una lista con los registros introducidos.
- Ordenación de los elementos almacenados.
- Eliminación completa de todos los registros.
- Actualización de la interfaz sin necesidad de recargar la página.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript

## Conceptos practicados

- Manipulación del DOM.
- Gestión de eventos.
- Uso de arrays y objetos.
- Validación básica de formularios.
- Actualización dinámica de contenido.
- Organización de código JavaScript.

## Objetivos del proyecto

- Consolidar conocimientos de JavaScript.
- Comprender el funcionamiento de aplicaciones interactivas en el navegador.
- Practicar la gestión y visualización dinámica de datos.
- Mejorar la capacidad de estructurar proyectos web frontend.

## Cómo ejecutar el proyecto

1. Clonar o descargar el repositorio.
2. Abrir el archivo `index.html` en cualquier navegador web moderno.
3. Introducir los datos de una mascota y comenzar a gestionar los registros.

## Posibles mejoras futuras

- Permitir la edición individual de registros.
- Implementar filtros de búsqueda por nombre o número de chip.
- Almacenar la información utilizando Local Storage para conservar los datos entre sesiones.
- Incorporar validaciones más avanzadas para los formularios.
- Añadir opciones de exportación de los registros.
